# CirPay

A USDC-based payment and finance platform with natural language processing capabilities.

🌐 **Live Site**: [https://cirpay.app](https://cirpay.app)

## About

CirPay is a Web3 payment application that enables users to send payments, create invoices, process payroll in USDC (and other stablecoins), and perform all these operations through natural language commands via an AI-powered agent.

## Features

- 🤖 **ARC Agent** - Natural language transaction processing
- 🎁 **Send Gift** - Quick payments with predefined amounts
- 🧾 **Invoice** - Invoice creation and tracking
- 💰 **Payroll** - Bulk salary payments
- 📜 **History** - Complete transaction records
- 👤 **Profile** - On-chain identity and portfolio
- 🏆 **Leaderboard** - User rankings by points
- ✅ **Tasks** - Gamification and rewards system

- **Framework**: Next.js 16
- **Blockchain**: Ethereum (via Wagmi & Viem)
- **Wallet**: RainbowKit
- **Database**: Supabase
- **AI**: GROQ SDK
- **Styling**: Tailwind CSS
